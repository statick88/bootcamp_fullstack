M√≥dulos en Python

Introducci√≥n a M√≥dulos.

Los m√≥dulos en python son archivos que contienen definiciones y declaraciones de python. Los m√≥dulos permiten organizar el c√≥digo en archivos separados. Los m√≥dulos se utilizan para reutilizar c√≥digo y para mantener el c√≥digo organizado.

Ejemplo:

# modulo.py 
def saludar():
    print("Hola Mundo")

En el c√≥digo anterior se define un m√≥dulo modulo.py que contiene una funci√≥n saludar.

Ejemplo:

# modulo.py
def despedir():
    print("Adi√≥s Mundo")

En el c√≥digo anterior se define un m√≥dulo modulo.py que contiene una funci√≥n despedir.

Creando el primer M√≥dulo.

Para crear nuestro primer m√≥dulo en python, creamos un archivo con extensi√≥n .py y definimos las funciones que queremos exportar.

Ejemplo:

# modulo_saludar.py

def saludar(nombre):
    print(f"Hola {nombre}")

En el c√≥digo anterior se define un m√≥dulo modulo_saludar.py que contiene una funci√≥n saludar.

Creando el segundo M√≥dulo.

Para crear nuestro segundo m√≥dulo en python, creamos un archivo con extensi√≥n .py y definimos las funciones que queremos exportar.

Ejemplo:

# modulo_despedir.py

def despedir(nombre):
    print(f"Adi√≥s {nombre}")

En el c√≥digo anterior se define un m√≥dulo modulo_despedir.py que contiene una funci√≥n despedir.

Creando el archivo principal.

Para utilizar los m√≥dulos en python, creamos un archivo principal con extensi√≥n .py e importamos los m√≥dulos que queremos utilizar.

Ejemplo:

# main.py
import modulo_saludar
import modulo_despedir

__name__ == "__main__"
modulo_saludar.saludar("Juan")
modulo_despedir.despedir("Juan")

En el c√≥digo anterior se importan los m√≥dulos modulo_saludar y modulo_despedir y se utilizan las funciones saludar y despedir.

Importando M√≥dulos.

Para importar un m√≥dulo en python se utiliza la palabra clave import seguida del nombre del m√≥dulo.

Tip

Utilizaremos el mismo ejemplo anterior.

Ejemplo:

# main.py

import modulo_saludar
import modulo_despedir

modulo_saludar.saludar("Juan")
modulo_despedir.despedir("Juan")

En el c√≥digo anterior se importan los m√≥dulos modulo_saludar y modulo_despedir y se utilizan las funciones saludar y despedir.

Renombrando M√≥dulos.

Para renombrar un m√≥dulo en python se utiliza la palabra clave as seguida del nuevo nombre.

Ejemplo:

# main.py

import modulo_saludar as saludar
import modulo_despedir as despedir

saludar.saludar("Juan")
despedir.despedir("Juan")

En el c√≥digo anterior se importan los m√≥dulos modulo_saludar y modulo_despedir con los nombres saludar y despedir respectivamente.

Seleccionando Elementos

Para importar elementos espec√≠ficos de un m√≥dulo en python se utiliza la palabra clave from seguida del nombre del m√≥dulo y la palabra clave import seguida del nombre del elemento.

Ejemplo:

# main.py

from modulo_saludar import saludar

saludar("Juan")

from modulo_despedir import despedir

despedir("Juan")

En el c√≥digo anterior se importan las funciones saludar y despedir del m√≥dulo modulo_saludar y modulo_despedir respectivamente.

Instalando M√≥dulos con pip

Para instalar m√≥dulos en python se utiliza la herramienta pip. pip es un sistema de gesti√≥n de paquetes utilizado para instalar y administrar paquetes de software escritos en python.

Ejemplo:

En el c√≥digo anterior se instala el m√≥dulo numpy utilizando pip.

Ahora pra utilizar este m√≥dulo en nuestro c√≥digo, simplemente lo importamos.
